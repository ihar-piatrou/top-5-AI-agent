# Database Schema — Top5Agent

## Overview

7 tables. The core flow goes:
`pipeline_runs` → `ideas` → `scripts` → `script_sections` → `media_assets`
                                       → `script_reviews`
                                       → `sources`

---

## Entity Relationship Diagram

```
pipeline_runs
    (standalone — tracks each run of the pipeline)

ideas
    ↓ 1:many
scripts
    ↓ 1:many              ↓ 1:many         ↓ 1:many
script_sections       script_reviews      sources
    ↓ 1:many
media_assets
```

---

## Tables

### `pipeline_runs`
Tracks every execution of the pipeline (manual or scheduled).

| Column           | Type              | Notes                              |
|------------------|-------------------|------------------------------------|
| `id`             | UNIQUEIDENTIFIER  | PK, `NEWSEQUENTIALID()`            |
| `trigger_reason` | NVARCHAR(200)     | e.g. "Manual run for niche 'cars'" |
| `status`         | NVARCHAR(50)      | `running` / `completed` / `failed` |
| `created_at`     | DATETIME2         | `GETUTCDATE()`                     |
| `completed_at`   | DATETIME2         | nullable                           |

---

### `ideas`
One row per video idea generated by GPT-4o.

| Column      | Type              | Notes                                              |
|-------------|-------------------|----------------------------------------------------|
| `id`        | UNIQUEIDENTIFIER  | PK, `NEWSEQUENTIALID()`                            |
| `title`     | NVARCHAR(300)     | Required. e.g. "5 Things Mechanics Say Never Do"   |
| `niche`     | NVARCHAR(100)     | e.g. "cars and mechanics"                          |
| `summary`   | NVARCHAR(1000)    | Short description used as script writer input      |
| `embedding` | NVARCHAR(MAX)     | JSON float[1536] from text-embedding-3-small       |
| `status`    | NVARCHAR(50)      | `pending` / `approved` / `rejected` / `scripted`  |
| `created_at`| DATETIME2         | `GETUTCDATE()`                                     |

**Relationships:**
- Has many `scripts`

---

### `scripts`
Full script produced by Claude for an approved idea.

| Column        | Type             | Notes                                           |
|---------------|------------------|-------------------------------------------------|
| `id`          | UNIQUEIDENTIFIER | PK, `NEWSEQUENTIALID()`                         |
| `idea_id`     | UNIQUEIDENTIFIER | FK → `ideas.id`                                 |
| `json_content`| NVARCHAR(MAX)    | Full ScriptJson stored as raw JSON              |
| `raw_text`    | NVARCHAR(MAX)    | Flattened readable version (optional)           |
| `status`      | NVARCHAR(50)     | `draft` / `reviewed` / `polished` / `approved` / `needs_review` |
| `created_at`  | DATETIME2        | `GETUTCDATE()`                                  |

**Relationships:**
- Belongs to one `idea`
- Has many `script_sections`
- Has many `script_reviews`
- Has many `sources`

---

### `script_sections`
One row per section of a script: hook (position 0), items 1–5, outro (position 99).

| Column       | Type             | Notes                                      |
|--------------|------------------|--------------------------------------------|
| `id`         | UNIQUEIDENTIFIER | PK, `NEWSEQUENTIALID()`                    |
| `script_id`  | UNIQUEIDENTIFIER | FK → `scripts.id`                          |
| `position`   | INT              | 0=Hook, 1–5=Items, 99=Outro               |
| `title`      | NVARCHAR(300)    | Section heading (null for hook/outro)      |
| `narration`  | NVARCHAR(MAX)    | Spoken narration text. Required.           |
| `media_query`| NVARCHAR(300)    | Stock footage search term for Pexels       |
| `media_type` | NVARCHAR(20)     | `photo` or `video`                         |
| `created_at` | DATETIME2        | `GETUTCDATE()`                             |

**Relationships:**
- Belongs to one `script`
- Has many `media_assets`

---

### `script_reviews`
Fact-check review produced by GPT-4o for a script.

| Column        | Type             | Notes                                      |
|---------------|------------------|--------------------------------------------|
| `id`          | UNIQUEIDENTIFIER | PK, `NEWSEQUENTIALID()`                    |
| `script_id`   | UNIQUEIDENTIFIER | FK → `scripts.id`                          |
| `reviewer`    | NVARCHAR(50)     | `gpt-4o` or `claude-sonnet-4-6`            |
| `review_text` | NVARCHAR(MAX)    | Human-readable summary of review           |
| `issues_found`| NVARCHAR(MAX)    | JSON array of FactCheckResult objects      |
| `approved`    | BIT              | 1 if all claims passed, 0 otherwise        |
| `created_at`  | DATETIME2        | `GETUTCDATE()`                             |

**Relationships:**
- Belongs to one `script`

---

### `sources`
Source URLs found by the fact reviewer for each script.

| Column      | Type             | Notes                                           |
|-------------|------------------|-------------------------------------------------|
| `id`        | UNIQUEIDENTIFIER | PK, `NEWSEQUENTIALID()`                         |
| `script_id` | UNIQUEIDENTIFIER | FK → `scripts.id`                               |
| `url`       | NVARCHAR(2000)   | Required. Source URL from fact review.          |
| `title`     | NVARCHAR(500)    | Source title (e.g. "CDC — Heat Stroke Facts")   |
| `verified`  | BIT              | 0 = unverified (default), 1 = manually verified |
| `created_at`| DATETIME2        | `GETUTCDATE()`                                  |

**Relationships:**
- Belongs to one `script`

---

### `media_assets`
Downloaded media files from Pexels for a script section.

| Column             | Type             | Notes                                         |
|--------------------|------------------|-----------------------------------------------|
| `id`               | UNIQUEIDENTIFIER | PK, `NEWSEQUENTIALID()`                       |
| `script_section_id`| UNIQUEIDENTIFIER | FK → `script_sections.id`                     |
| `pexels_id`        | NVARCHAR(50)     | Pexels photo or video ID                      |
| `asset_type`       | NVARCHAR(20)     | `photo` or `video`. Required.                 |
| `remote_url`       | NVARCHAR(2000)   | Original Pexels URL. Required.                |
| `local_path`       | NVARCHAR(1000)   | Local file path e.g. `media/{runId}/{sectionId}/photo_1.jpg` |
| `attribution`      | NVARCHAR(500)    | e.g. "Photo by John Doe on Pexels"            |
| `created_at`       | DATETIME2        | `GETUTCDATE()`                                |

**Relationships:**
- Belongs to one `script_section`

## Full Relationship Summary

```
pipeline_runs (no FK — standalone run tracker)

ideas
  └── scripts (idea_id → ideas.id)
        ├── script_sections (script_id → scripts.id)
        │     └── media_assets (script_section_id → script_sections.id)
        ├── script_reviews  (script_id → scripts.id)
        └── sources         (script_id → scripts.id)
```

---

## Status Lifecycle

### `ideas.status`
```
pending → approved → scripted
       → rejected
```

### `scripts.status`
```
draft → reviewed → polished → approved
              → needs_review
```

### `pipeline_runs.status`
```
running → completed
        → failed
```

---

## Notes

- All PKs use `NEWSEQUENTIALID()` — sequential GUIDs for better index performance vs random `NEWID()`
- All timestamps use `GETUTCDATE()` — stored in UTC
- `ideas.embedding` stores a JSON-serialized `float[1536]` vector used for cosine similarity deduplication in C#
- `script_reviews.issues_found` stores a JSON array of `FactCheckResult` objects
- `sources.verified` defaults to `0` — GPT-sourced URLs are never auto-trusted; require manual verification
- `media_assets.attribution` is required by the Pexels license ("Photo/Video by X on Pexels")
